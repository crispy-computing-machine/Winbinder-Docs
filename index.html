DISCLAIMER - Please read this

Although the basic structure of WinBinder is already defined, it is still in alpha stage. This means that several things are still subject to change, including specifications, function parameters, parameter order, function names, and more. It is very likely that you will have to change at least a small part of your source code in the future. Also note that this documentation is still incomplete and it is possible that some information is inaccurate. Use WinBinder at your own risk.

If you believe you have found an error in this documentation, please submit it to the WinBinder forum. Thank you for your support.
